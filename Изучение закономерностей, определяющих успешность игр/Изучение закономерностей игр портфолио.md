## Гипотезы

### Средние пользовательские рейтинги платформ Xbox One и PC одинаковые

Проверим, отличаются ли средние пользовательские рейтинги платформ Xbox One и PC. 
Нулевая гипотеза - "средние рейтинги не отличаются".
Альтернативная гипотеза - "Средние рейтинги отличаются"
Значение критического уровня вероятности (альфа) примем равным 5%.
я буду использовать метод: scipy.stats.ttest_ind (array1, array2, equal_var), поскольку это специальный метод для проверки гипотезы о равенстве среднего двух генеральных совокупностей по взятым из них выборкам.
Пропущенные значечния отфильтруем.



```python
alpha = 0.05
results = st.ttest_ind(
    data[(data['platform'] == "XOne") & (data['user_score'].notna())]['user_score'], 
    data[(data['platform'] == "PC") & (data['user_score'].notna())]['user_score']
)

print('p-значение:', results.pvalue)

if results.pvalue < alpha:
    print('Отвергаем нулевую гипотезу')
else:
    print('Не получилось отвергнуть нулевую гипотезу') 
 
```

    p-значение: 1.3037979620149106e-05
    Отвергаем нулевую гипотезу
    

Получается что статистически средние рейтинга на XOne и PC отличаются, о чем нам говорит p-значение (уровень значимости).

Проверим, отличаются ли средние пользовательские рейтинги жанров Action One и Sports. 
Нулевая гипотеза - "средние рейтинги не отличаются".
Альтернативная гипотеза - "Средние рейтинги отличаются"
Значение критического уровня вероятности (альфа) примем равным 5%.
я буду использовать метод: scipy.stats.ttest_ind (array1, array2, equal_var), поскольку это специальный метод для проверки гипотезы о равенстве среднего двух генеральных совокупностей по взятым из них выборкам.
Пропущенные значечния отфильтруем.


```python
alpha = 0.05 
results = st.ttest_ind(
    data[(data['genre'] == "Action") & (data['user_score'].notna())]['user_score'], 
    data[(data['genre'] == "Sports") & (data['user_score'].notna())]['user_score']
)

print('p-значение:', results.pvalue)

if results.pvalue < alpha:
    print('Отвергаем нулевую гипотезу')
else:
    print('Не получилось отвергнуть нулевую гипотезу') 
```

### Промежуточный итог
